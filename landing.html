Modules

Four Live API Integrations

Each module is self-contained, documented, and driven by a real third-party API

Geospatial Intelligence

Search any location in Jamaica — Leaflet flies the map and drops an animated pin marker with a popup.

Real-Time News Stream

Searches the New York Times Article Search API and renders results as animated glassmorphism cards.

NYTimes Article Search API

On-Demand Media Player

Fetches the top 5 YouTube videos for any query via template literals and plays them in an embedded iframe.

YouTube Data API v3

State-Sync Analog Clock

A 60fps analog clock drawn with Math.cos / Math.sin trigonometry. Toggles between Day and Night themes without pausing the rAF loop.

HTML5 Canvas · requestAnimationFrame
